# Resource Selection (Probability) Functions for Use-Availability Data

Resource Selection (Probability) Functions for use-availability wildlife
data based on weighted distributions as described in Lele and Keim
(2006), Lele (2009), and Solymos & Lele (2016).

## Details

[`rsf`](rsf.md): Resource Selection Functions (RSF)

[`rspf`](rsf.md): Resource Selection Probability Functions (RSPF)

[`hoslem.test`](hoslem.test.md): Hosmer-Lemeshow Goodness of Fit (GOF)
Test

Visual summaries: [`kdepairs`](kdepairs.md) for 2D scatterplots and
[`mep`](mep.md) for marginal effect plots based on fitted model objects.
